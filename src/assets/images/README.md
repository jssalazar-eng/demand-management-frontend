# Logo Placeholder

Coloca aquí tu archivo `logo.webp` con el logo de HACEB.

## Especificaciones recomendadas:

- Formato: WebP
- Tamaño: 512x512 px (o proporción cuadrada)
- Fondo: Transparente
- Resolución: Adecuada para pantallas de alta densidad

## Ubicación:

`src/assets/images/logo.webp`

## Uso:

El componente Logo importará automáticamente esta imagen y la mostrará en el header de la aplicación con fallback al logo generado por código si la imagen no está disponible.
